== Installation ==
1. Upload plugin folder to the /wp-content/plugins/ directory<br>
1. Activate the plugin through the 'Plugins' menu in WordPress<br>
1. Im Appearance->widgets select Twitter Only Widgets and add the User and the number of tweets you want to display in the sidebar. <br>
